Live Sports
===============================

version number: 0.0.1
author: Zach Manno

Overview
--------

Pulling live sports data from the MySportsFeeds API

Installation / Usage
--------------------

To install use pip:

    $ pip install livesports


Or clone the repo:

    $ git clone https://github.com/ZachManno/livesports.git
    $ python setup.py install
